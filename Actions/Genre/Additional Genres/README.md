# Additional Genres

Extended list of one-click `GENRE` tag setters. Each action rewrites the `GENRE` field on the selected files to a single, controlled value. Compound values (joined with `;`) tag a release with multiple sub-genres at once.

## Files

- `Additional Genres.json` — full action group as a single mp3tag-compatible JSON array.
- `*.mta` — individual mp3tag action scripts for each genre preset.
