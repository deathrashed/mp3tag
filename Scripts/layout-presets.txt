Music Library Layout Presets
============================

Purpose
-------

This document defines the recommended folder structure presets for the
interactive Mp3tag configuration wizard.

These layouts are reference implementations based on folder structures commonly
used by music collectors, Plex, Jellyfin, Navidrome, Kodi, MusicBee,
foobar2000, Roon, and large personal music libraries.

The wizard does not need to implement these layouts exactly as shown. Presets
may be renamed, refined, or expanded where appropriate, but each should remain
distinct and include a live ASCII preview.

Each preset should display:

• Layout name
• Short description
• Folder structure
• Live preview

[1] Standard Hierarchy (Recommended)

Description
    The classic music library structure used by most collectors.

Structure
    Artist / Album / TrackNo - Title

Preview

    /Music/
    └── Iron Maiden/
        └── Powerslave/
            └── 02 Aces High.flac

Implementation Notes
--------------------

• These presets should be presented as an interactive list.
• Every preset should include a live ASCII preview.
• Users should be able to move backwards without losing selections.
• The wizard should remember previous selections where appropriate.
• The Custom Layout Builder should allow users to generate layouts rather than
  being limited to predefined presets.
• The implementation may include additional layouts if they improve usability,
  provided the interface remains clear and easy to navigate.
